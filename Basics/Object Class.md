The object class is the base class from which all the classes ultimately inherit
Provides a minimal set of default methods that every object has.

# Default method provided
- `__init__ (self)` - Constructor
- `__str__(self)` - String Representation
- `__repr__(self)` - 
- `__eq__(self, other)` - Equality Check
- `__hash__(self)` - Hash value
- `__class__(self)` - Reference to the class of the object
