UV refers to a fast, unified package and project manager that streamlines common development workflows.

Aims to replace multiple other tools like `pip`, `venv`, `Poetry`, `pyenv` by providing a single command-line interface for installing Python versions creating virtual environments and installing package.

## Features of UV

- Dependency Management - `uv` can build and manage dependecies from `requirements.txt` and other modern formats and it uses lockfiles for reproducible builds.
